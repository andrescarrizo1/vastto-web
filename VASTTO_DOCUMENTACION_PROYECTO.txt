1. ¿QUÉ ES VASTTO?
-------------------
VASTTO (vastto.com.ar) es un sitio web nicho y comparador tecnológico enfocado en la monetización mediante Marketing de Afiliados (Mercado Libre Argentina).
Su modelo de negocio es generar tráfico orgánico (SEO) resolviendo intenciones de búsqueda informacionales y transaccionales ("Long-Tail Keywords"), para derivar dichas visitas hacia Mercado Libre mediante enlaces de afiliados.

- Dominio: vastto.com.ar
- Hosting: Cloudflare Pages (deploy automático desde GitHub main branch)
- Repositorio: https://github.com/andrescarrizo1/vastto-web
- Arquitectura Core: Astro 4.x (Static Site Generation / SSG)
- Código de afiliado ML: caan170047 (administrado centralmente en `src/utils/affiliate.ts`)

2. ESTRATEGIA SEO Y TOPICAL AUTHORITY (LA LEY DE VASTTO)
--------------------------------------
En vez de ser un clon de e-commerce, VASTTO ataca "El viaje del comprador".
Google penaliza los listados de precios vacíos. Para ganarle a Mercado Libre en las SERPs, VASTTO se posiciona como el EXPERTO que compara, analiza y decide.

Pilares del SEO de VASTTO:
A. Topical Authority (Silos): El contenido se agrupa en categorías estrictas: Celulares, Auriculares y Smartwatches. Cada silo tiene una página índice (`/celulares`) que transfiere autoridad ("Link Juice") a las guías profundas (`/celulares/samsung-a55-vs-motorola-edge-50-fusion`).
B. E-E-A-T (Experiencia, Expertise, Autoridad, Confianza): Para cumplir con el estándar de Google 2026, cada artículo incluye componentes estandarizados:
   - `AuthorBio.astro` (Muestra el rostro y la bio de Andrés).
   - `EditorialVerdict.astro` (Veredicto claro al final del artículo).
   - `AffiliateDisclaimer.astro` (Aviso legal de afiliación obligatorio).
C. Frecuencia de Actualización: Los precios en e-commerce caducan rápido. Implementamos un scraper (Puppeteer) para mantener todo actualizado (Ver sección 4).

3. HITOS DE DESARROLLO (ACTUALIZACIÓN 29/03/2026)
-----------------------------------
Durante nuestra última sesión maestra, se hizo un refactor de Nivel 2.0 en el sitio:

A) CORE WEB VITALS & TECHNICAL SEO (Puntuación 100/100):
   - Sitemap Dinámico: Se eliminó el `sitemap.xml` estático y se instaló el plugin `@astrojs/sitemap@3.1.6`. Ahora, el archivo `sitemap-index.xml` se autogenera en cada compilación escaneando todas nuestras rutas.
   - Prevención de CLS (Cumulative Layout Shift): Se reprogramó `LazyImage.astro` para forzar explícitamente dimensiones (width/height) y carga `decoding="async"`. La web ya no "salta" al cargar en 4G.
   - Robots.txt: Centralizado dinámicamente apuntando a nuestro nuevo sitemap.

B) MODO OSCURO GLOBAL (Dark Mode):
   - Se migró el CSS 100% estático a un sistema de tokens dinámicos en `global.css` (`--vastto-bg`, `--text-main`, etc.).
   - Se inyectó un script anti-FOUC (Flash of Unstyled Content) en `BaseHead.astro` para analizar el `localStorage` antes de renderizar el HTML.
   - Más de 360 archivos y componentes Astro fueron refactorizados por script para usar las nuevas variables.

C) EXPANSIÓN DE CONTENIDO:
   - Se amplió radicalmente la cobertura de guías en todos los silos.
   - Nuevos artículos "Killer":
     * `/celulares/samsung-a55-vs-motorola-edge-50-fusion`
     * `/auriculares/sony-wh-ch520-review`
     * `/smartwatch/kieslect-ks-pro-review`

   - Se implementó un "Menú Hamburguesa" móvil en `Header.astro` para mejorar el Crawleo de Googlebot por los silos.

4. AUTOMATIZACIÓN DE PRECIOS (PUPPETEER)
---------------------------------------------------
Dado que la inflación/cambio de precios de ML afecta el CTR (Click Through Rate), construimos motores de scraping en la carpeta `scripts/`.
- Archivo prinicpal: `scripts/update-prices.js`
- Tecnología: `puppeteer-extra` con `stealth-plugin`.
- Funcionamiento: Recorre los componentes de Astro, lee las URLs de Mercado Libre, scrapea el DOM estático extraído, actualiza el precio dentro del `.astro` y guarda el archivo.
- Uso operativo: Ejecutar `npm run update-prices` localmente, revisar los `git diff`, y hacer un `git push`. (Nota: no ejecutar en exceso para evitar baneos de IP de ML).
